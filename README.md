TD API: 

C'est une API fait via TyepScript, son fonctionnement est simple : L'api est herberger localement et a deux "fonction" bien spécifique, un get qui permet de récupérer les users qui sont dans l'API et un POST qui permet lui de stocker des utilisteur dans l'array "users".
